# Get Involved

Hyperlight is a [Cloud Native Computing Foundation](https://www.cncf.io/) sandbox project with an active and welcoming community.

### CNCF Slack

Join the [#hyperlight channel](https://cloud-native.slack.com/archives/C08181YF16K) on CNCF Slack for real-time discussion with maintainers and users.

New to CNCF Slack? [Get an invite here](https://communityinviter.com/apps/cloud-native/cncf).

### Community Meetings

We hold fortnightly community meetings open to everyone:

- **When:** Wednesdays 09:00 PST (every two weeks)
- **Where:** [Zoom](https://zoom.us/j/91509780029)
- **Agenda:** [Hackmd](https://hackmd.io/blCrncfOSEuqSbRVT9KYkg)

<Aside type="tip">If you have a great idea for a new feature, come discuss it in one of fortnightly meetings. You will
be welcomed and we'll be interested in hearing what you have to say.</Aside>

## Contributing

We welcome contributions of all kinds:

- **Code** — Bug fixes, features, and improvements
- **Documentation** — Tutorials, guides, and clarifications
- **Community** — Answering questions, reviewing PRs

See the [Contributing Guide](https://github.com/hyperlight-dev/hyperlight/blob/main/CONTRIBUTING.md) to get started.

## Code of Conduct

Hyperlight follows the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md). We are committed to providing a welcoming and inclusive community.

## Maintainers

The project is maintained by contributors from Microsoft and the broader community. See the [MAINTAINERS.md](https://github.com/hyperlight-dev/hyperlight/blob/main/MAINTAINERS.md) file for the current list.
