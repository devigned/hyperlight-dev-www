Welcome to the Hyperlight documentation. Hyperlight is a lightweight Virtual Machine Manager (VMM) designed to run hardware-isolated micro-VMs with single-digit millisecond cold starts.

## Getting Started

New to Hyperlight? Start with the [Getting Started](/docs/getting-started/) guide to learn the basics.

## Ecosystem

Explore the Hyperlight ecosystem:

- [Hyperlight Wasm](/docs/ecosystem/hyperlight-wasm/) — WebAssembly support via the Component Model
- [Hyperlight Nanvix](/docs/ecosystem/hyperlight-nanvix/) — POSIX compatibility with 150+ syscalls
